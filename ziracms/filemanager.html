Работа с файлами

Zira CMS имеет файловый менеджер с широкими возможностями.

С помощью редактора изображений, вы можете изменять длину, ширину, пропорции картинок, обрезать часть картинки, вставлять водяной знак итд.

Кроме того, в Zira CMS встроен редактор HTML-кода TinyMCE, с помощью которого можно редактировать HTML-файлы.

 

Загрузка файлов и создание папок через файловый менеджер осуществляется очень просто.

Для загрузки файла (или файлов) на сервер нужно нажать на кнопку "Загрузить" в левой колонке окна. После чего, вы увидите диалоговое окно в котором нужно выбрать загружаемый файл. Для загрузки сразу нескольких файлов нужно выделять их удерживая клавину ctrl (комбинация может быть другой в вашей операционной системе). После выбора файлов и нажатия подтверждающей кнопки должна начаться загрузка на сервер, при этом вы увидите окно отображающее процесс загрузки.

 

Кроме того, загрузить файл на сервер можно, также методом "Drag and Drop", перетащив и бросив файл в окно файлового менеджера Zira CMS.

Надо сказать, что метод "Drag and Drop" поддерживают различные окна в Zira CMS. Например, можно перетащить изображение из файлового менеджера в галерею или слайдер. При перетаскивании файлов между двумя окнами файлового менеджера можно копировать файлы из одной папки в другую.

 

Но это не единственный способ загрузки файлов в Zira CMS. Также, можно загрузить файл из адреса URL "напрямую", не скачивая его себе на компьютер. Для этого достаточно выбрать соответствующий пункт меню и ввести полный адрес к файлу.

Кроме того, если ваш сервер поддерживает ZIP, вы можете загрузить архив с файлами на сервер и распаковать его в файловом менеджере.

 

При двойном клике по иконке файла открывается окно редактирования файла. В зависимости от типа файла, это может быть редактор изображений, HTML-редактор, либо текстовый редактор, типа блокнота.

 

Следует иметь в виду, что при редактировании изображений с очень высоким разрешением, может возникать ошибка связанная с нехваткой памяти выделенной для PHP на сервере. В этом случае, нужно проконсультироваться у хостинг-провайдера по вопросу возможности её увеличения.

 

Максимальный размер загружаемого файла зависит от настройки PHP на сервере. Каждый хостинг провайдер выставляет лимит по своему усмотрению, но в большинстве случаев его можно увеличить.
