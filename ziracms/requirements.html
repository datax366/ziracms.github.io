Системные требования Zira CMS

Перед установкой Zira CMS необходимо ознакомиться с системными требованиями и подобрать соответствующий хостинг, либо отдельный сервер.

 

Для работы Zira CMS требуются:

  • веб-сервер Apache
  • база данных MySQL 5 или SQLite 3
  • PHP 5.5.3 и выше
  • PDO
  • OpenSSL
  • GD

Для обеспечения максимальной быстроты и стабильности работы рекомендуется выбирать сервер на операционной системе семейства UNIX, таких как Linux, FreeBSD итд. Тем не менее, в ознакомительных целях, допускается установка Zira CMS на WIndows.

 

В связи с тем, что оконный интерфейс Zira CMS использует новейшие функции и возможности языка JavaScript, рекомендуется использовать последнюю версию браузера Mozilla Firefox, либо Google Chrome для управления системой.

Браузер Microsoft Internet Explorer поддерживается только для фронтенда и использовать его для администрирования не желательно.
